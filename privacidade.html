Política de Privacidade

Última atualização: maio de 2026

A Liga Local respeita a tua privacidade e está empenhada em proteger os teus dados pessoais de acordo com o Regulamento Geral sobre a Proteção de Dados (RGPD) — Regulamento (UE) 2016/679.

1. Quem somos

Liga Local é uma plataforma de organização de jogos e eventos desportivos amadores na região de Aveiro, Portugal. Para questões relacionadas com privacidade, podes contactar-nos pelo email: hello@ligalocal.pt

2. Dados que recolhemos

Recolhemos apenas os dados necessários para te enviar a newsletter e organizar os jogos na tua zona:

  • Nome próprio (opcional)
  • Endereço de email
  • Localização (zona de Aveiro)
  • Preferências desportivas (modalidades, nível, disponibilidade)
  • Consentimentos dados por ti

3. Base legal para o tratamento

Os teus dados são tratados com base no teu consentimento explícito, dado no momento da subscrição (artigo 6.º, n.º 1, alínea a) do RGPD). Podes retirar esse consentimento a qualquer momento.

4. Para que usamos os teus dados

  • Enviar a newsletter semanal com eventos e jogos na tua zona
  • Personalizar o conteúdo de acordo com as tuas modalidades e localização
  • Enviar comunicações de parceiros, apenas se tiveres dado consentimento

5. Quanto tempo guardamos os teus dados

Guardamos os teus dados enquanto mantiveres a subscrição ativa. Após a cancelação, os dados são eliminados no prazo máximo de 30 dias.

6. Partilha com terceiros

Não vendemos nem partilhamos os teus dados com terceiros para fins comerciais. Utilizamos o serviço EmailJS apenas para o envio de emails transacionais, sob as garantias de privacidade do respetivo fornecedor.

7. Os teus direitos

Ao abrigo do RGPD, tens os seguintes direitos:

  • Acesso — saber que dados temos sobre ti
  • Retificação — corrigir dados incorretos
  • Eliminação — pedir a remoção dos teus dados
  • Portabilidade — receber os teus dados em formato legível
  • Oposição — opor-te ao tratamento para fins de marketing
  • Retirada do consentimento — a qualquer momento, sem prejuízo

Para exercer qualquer um destes direitos, contacta-nos: hello@ligalocal.pt

8. Cookies

Este site não utiliza cookies de rastreamento nem ferramentas de análise de terceiros. Os dados do formulário são guardados temporariamente no teu browser (localStorage) apenas para facilitar a navegação entre etapas e são eliminados após a subscrição.

9. Contacto e reclamações

Se tiveres questões sobre como tratamos os teus dados, contacta-nos em hello@ligalocal.pt. Tens também o direito de apresentar reclamação à autoridade de controlo nacional: CNPD — Comissão Nacional de Proteção de Dados (www.cnpd.pt).
